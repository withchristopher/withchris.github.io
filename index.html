FAQ

  • What is the MERN Stack?

    MERN stack is a web development framework. It consists of MongoDB, ExpressJS, ReactJS, and NodeJS as its working components. Here are the details of what each of these components is used for in developing a web application when using MERN stack:

    MongoDB: A document-oriented, No-SQL database used to store the application data.

    NodeJS: The JavaScript runtime environment. It is used to run JavaScript on a machine rather than in a browser.

    ExpressJS: A framework layered on top of NodeJS, used to build the backend of a site using NodeJS functions and structures. Since NodeJS was not developed to make websites but rather run JavaScript on a machine, ExpressJS was developed.

    ReactJS: A library created by Facebook. It is used to build UI components that create the user interface of the single page web application.

  • Do you build mobile friendly applications?

    Full stack web applications need to be mobile-first. Designed as a security first web applicaitons, the goal is to deliver content in your style to any device, Operating Systems and screen size.

    The MERN stack provides these capabilities is a secure and performant way. Bringing both online and offline capabilities to users, your application should be easily accessible as well as downloadable to your device as a Progressive Web Application (PWA).
